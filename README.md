Data Analytics Project – Customer Shopping Behavior
1. Overview

  This project analyzes customer shopping behavior using data analytics techniques. The dataset is processed using Python for exploratory data analysis (EDA) and data cleaning. SQL queries are executed on PostgreSQL to extract insights, and the results are visualized through a Power BI dashboard. A final report and presentation are created to summarize the findings.

2. Dataset

 The dataset contains information about customer purchases, including:

  . Customer ID

  . Age

  . Gender

  . Item Purchased

  . Category

  . Purchase Amount

  . Review Rating

  . Frequency of Purchases

  . Discount Applied

  . Previous Purchases

  . The dataset helps analyze purchasing patterns and customer behavior.

3. Tools & Technologies

 The following tools and technologies were used in this project:

  . Python – Data processing and analysis

  . Pandas & NumPy – Data cleaning and transformation

  . PostgreSQL / MySQL / SQL Server – Running SQL queries and data storage

  . Power BI – Data visualization and dashboard creation

  . PowerPoint / Gamma – Presentation of results

4. Project Steps
 1. Data Loading

  . Loaded the dataset using Python (Pandas).

  . Imported the data into PostgreSQL database.

 2. Data Cleaning

  . Handled missing values.

  . Converted categorical data into useful formats.

  . Created new features such as age groups and purchase frequency in days.

 3. Exploratory Data Analysis (EDA)

  . Analyzed customer demographics.

  . Studied purchasing patterns across categories.

  . Identified trends in discounts and review ratings.

 4. SQL Analysis

 Several SQL queries were used to extract insights such as:

  . Top purchased items by category

  . Products with highest discounts

  . Customer segmentation based on previous purchases

  . Average review ratings

 5. Dashboard Creation

 A Power BI dashboard was created to visualize insights including:

  . Sales by category

  . Purchase frequency

  . Customer segments

  . Discount impact on purchases

 5. Dashboard Features

 The Power BI dashboard includes:

  . Category-wise sales analysis

  . Customer age group analysis

  . Discount usage insights

  . Product popularity analysis

  . Interactive filters and charts

 6. Results & Insights

 Key findings from the analysis include:

  . Certain categories have significantly higher purchase frequency.

  . Discounts influence customer buying behavior.

  . Returning and loyal customers contribute significantly to sales.

  . Some products receive consistently higher review ratings.

 7. How to Run the Project

 Install required Python libraries:

  . pip install pandas numpy sqlalchemy psycopg2

  . Load the dataset using Python.

  . Import the data into PostgreSQL.

  . Run SQL queries for analysis.

  . Open the Power BI file to view the dashboard.

  . Review the final report and presentation.

 8. Author

 This project was created as part of a Data Analytics project to analyze customer shopping behavior and generate actionable insights using Python, SQL, and Power BI.
